About

The Demographic Statistics website provides valuable insights into various demographic indicators and statistics related to population dynamics. It is based on a comprehensive dataset that encompasses a wide range of demographic factors.

The dataset used in this project includes information such as birth rates, life expectancy, literacy rates, mortality rates, and government investment in different regions and countries. The data is collected from WorldBank.

To visualize and analyze the dataset, I have utilized a powerful suite of data visualization and analytics tools from Tableau. These tools include Tableau Prep, Tableau Desktop, and Tableau Public. Tableau Prep is used for data preparation and cleaning, while Tableau Desktop enables us to create interactive and insightful visualizations. The final visualizations are published and shared using Tableau Public, allowing users to explore and interact with the data online.

We hope that this website serves as a valuable resource for exploring and understanding the demographic trends and statistics that shape our world.

Data Source

The visualizations on this website are based on a comprehensive dataset that includes demographic statistics. The dataset covers a wide range of indicators related to birth rate, life expectancy, literacy rate, mortality rate, and government investment in the healthcare sector.

The data source: World bank health nutrition and population statistics

About the dataset:

The dataset contains 6 columns and more than 58,000 rows after the prep work from tableau prep. The prep work included pivoting the columns to rows. The columns in queston were the years with their respective values. However this format was inconvinient for visualization and so, it has been changes

The dataset can be viewed by clicking on the following link: Dataset

The dataset contains the following attributes or parameters:

  • Year:The years for which the data is available
  • Values:The values for each country for each year
  • Series Name: Name of the topic, like birth rate or life expectancy
  • Series Code: Code for the topic
  • Country Name: Name of the country for which the data is available
  • Country Code: Code of the corresponding country
